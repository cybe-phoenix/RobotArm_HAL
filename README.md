# 基于 STM32 的六自由度机械臂示教与回放控制系统 V1.0

## 1. 项目简介

本项目基于 STM32F103C8T6 和 HAL 库实现六自由度机械臂的示教与回放控制。系统通过五路电位器采集机械臂关节目标位置，使用定时器 PWM 控制五个关节舵机和一个夹爪舵机，并通过四个独立按键实现动作点记录、动作序列回放、夹爪开合和记录清空等功能。

当前版本已经完成无 OLED 条件下的核心功能，包括：五关节实时控制、动作点记录、线性插值回放、夹爪状态同步记录与回放、记录清空保护、模块化代码整理和主要代码注释。

## 2. 硬件平台

- 主控芯片：STM32F103C8T6
- 开发环境：STM32CubeMX + Keil MDK
- 软件库：STM32 HAL 库
- 舵机型号：MG90S、MG996R
- 输入设备：五个电位器、四个独立按键
- 输出设备：五个关节舵机、一个夹爪舵机

## 3. 功能说明

- 五个电位器实时控制机械臂五个关节。
- KEY3 控制夹爪打开与闭合。
- KEY4 记录当前机械臂动作点。
- KEY2 回放已记录的动作序列。
- KEY1 清空已记录的动作点。
- 每个动作点保存五路关节 ADC 值和一个夹爪状态。
- 回放时，相邻动作点之间采用线性插值，实现平滑过渡。
- 夹爪状态可以随动作点同步记录和回放。
- 支持调试变量开关 `ROBOT_DEBUG_ENABLE`，便于开发和正式版本切换。

## 6. 模块说明

### 6.1 `robot_servo`

舵机控制模块，主要负责五个关节舵机的 PWM 控制。

主要功能包括：

- 启动五个关节舵机 PWM 输出。
- 将五个关节舵机设置到中位。
- 将 ADC 采样值映射为 PWM 值。
- 对 PWM 值进行安全限幅。
- 根据五路 ADC 实时更新关节舵机位置。
- 在回放过程中同步设置五路关节舵机 PWM 值。

### 6.2 `robot_gripper`

夹爪控制模块，主要负责夹爪舵机的打开与闭合。

主要功能包括：

- 限制夹爪 PWM 输出范围。
- 控制夹爪打开。
- 控制夹爪闭合。
- 输出指定时间后停止夹爪 PWM，减少舵机持续受力发热。

### 6.3 `robot_key`

按键扫描模块，主要负责四个独立按键的读取和消抖。

按键采用低电平有效方式：

- KEY1：清空记录。
- KEY2：回放动作。
- KEY3：夹爪开合。
- KEY4：记录动作点。

### 6.4 `robot_record`

动作记录与回放模块，主要负责示教点保存和动作序列回放。

主要功能包括：

- 保存当前五路 ADC 值。
- 保存当前夹爪状态。
- 清空已记录动作点。
- 判断记录点数量是否满足回放条件。
- 对相邻动作点进行线性插值。
- 回放过程中同步恢复夹爪状态。

### 6.5 `main.c`

主流程文件，主要负责系统初始化和各模块调度。

主要流程包括：

1. 初始化 HAL、系统时钟、GPIO、DMA、ADC、TIM。
2. 设置舵机中位并启动 PWM。
3. 启动 ADC + DMA 连续采集。
4. 在主循环中扫描按键。
5. 根据按键调用对应模块函数。
6. 非回放状态下，根据电位器实时控制机械臂关节。

## 7. 使用方法

1. 上电后，机械臂五个关节进入初始中位。
2. 旋转五个电位器，实时调整机械臂姿态。
3. 按下 KEY4，记录当前动作点。
4. 多次调整姿态并按 KEY4，可记录多个动作点。
5. 按下 KEY3，可切换夹爪打开或闭合状态。
6. 按下 KEY2，机械臂按照记录顺序回放动作。
7. 按下 KEY1，清空已记录的所有动作点。
8. 清空后再次按 KEY2，系统不会产生动作变化。

## 8. 回放逻辑说明

系统采用动作点序列回放方式。每个动作点包含：

- 五个关节对应的 ADC 采样值。
- 当前夹爪状态。

当记录点数量小于 2 时，系统不执行回放。当记录点数量大于等于 2 时，系统按照如下方式依次回放：

```text
点 0 -> 点 1
点 1 -> 点 2
点 2 -> 点 3
...
```

每两个相邻动作点之间通过线性插值生成中间 PWM 值，使舵机动作更加平滑。

## 9. 调试开关

工程中设置了调试宏：

```c
#define ROBOT_DEBUG_ENABLE  0
```

当设置为 `1` 时，启用调试变量，便于在 Keil Watch 窗口中观察 ADC、按键和回放状态。

当设置为 `0` 时，关闭调试变量，使代码更接近正式运行版本。
